# API_Flask
REST API, предоставляющий возможность ведения микроблога.

API имеет 2 сущности:
  Пользователь,
  Пост.

Пользователь имеет возможность:
  создать,
  прочитать,
  изменить,
  удалить пост.
  
Задание выполнено с помощью фреймворка Flask.
